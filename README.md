Smart Meal Planner is a responsive web-based application designed to help users organize weekly meals, track calorie intake, and plan personalized nutrition goals.

The system allows structured meal planning for breakfast, lunch, and dinner across all seven days while supporting calorie tracking and dietary goal management such as:

Weight loss

Weight gain

High-protein diet

Maintenance diet
